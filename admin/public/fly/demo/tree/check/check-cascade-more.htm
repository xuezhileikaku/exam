Advanced - Cascade check nodes
View code

Select the parent affect the child, child choice does not affect the parent（父级选择影响子级，子级选择不影响父级）:
new fly.mini.Tree({
        checkMode: fly.mini.checkMode.multi,
        checkCascade: {
                check: { parent: false, children: true },
                uncheck: { parent: false, children: true }
        }
});


Select the affected parent the child, parent choice does not affect children（子级选择影响父级，父级选择不影响子级）:
new fly.mini.Tree({
        checkMode: fly.mini.checkMode.multi,
        checkCascade: {
                check: { parent: true, children: false },
                uncheck: { parent: true, children: false }
        }
});


Affects only the most recent one layer（只影响最近的一级）:
new fly.mini.Tree({
        checkMode: fly.mini.checkMode.multi,
        checkCascade: {
                check: { parent: 1, children: 1 },
                uncheck: { parent: 1, children: 1 }
        }
});

Or:
new fly.mini.Tree({
        checkMode: fly.mini.checkMode.multi,
        checkCascade:1
});


You can also（你还可以）:
Affect the parent and child（影响父级和子级）:
new fly.mini.Tree({
        checkCascade:true
});

Equivalent to（等同于）:
new fly.mini.Tree({
        checkCascade: {
                check: { parent: true, children: true },
                uncheck: { parent: true, children: true }
        }
})
Parent and child do not affect(父级和子级都不影响):
new fly.mini.Tree({
        checkCascade:false
});

Equivalent to（等同于）:
new fly.mini.Tree({
        checkCascade: {
                check: { parent: false, children: false },
                uncheck: { parent: false, children: false }
        }
});

Equivalent to（等同于）:
new fly.mini.Tree({
        checkCascade:0
});
Affect the three layer parent and three layer child(影响3层父级和3层子级):
new fly.mini.Tree({
        checkCascade:3
})

Equivalent to（等同于）:
new fly.mini.Tree({
        checkCascade: {
                check: { parent: 3, children: 3 },
                uncheck: { parent: 3, children: 3 }
        }
})
Affect the one layer parent and three layer child(影响1层父级和3层子级):
new fly.mini.Tree({
        checkCascade: {
                check: { parent: 1, children: 3 },
                uncheck: { parent: 1, children: 3 }
        }
})
